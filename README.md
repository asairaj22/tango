# Tango

## Technologies

* Electron - version 1.x.x (1.7.6)
* IDE - Any
* NPM - 3.10.10
* node - v6.11.3

## Installation steps

Install the dependencies

```sh
$ npm install
```

```sh
$ bower install
```

To run as electron

```sh
$ gulp run
```

for live re-load

```sh
$ gulp live
```

to run the app independently (just for info, we won't need this)

```sh
$ npm start
```


### To Package & Build (to generate exe/app)

For Windows

```sh
$ npm run package-win
```

For Mac

```sh
$ npm run package-mac
```

For Linux

```sh
$ npm run package-linux
```
